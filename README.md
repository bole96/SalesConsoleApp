# Sales Console Application (C# OOP Project)

## 📌 Overview
This is a simple console-based sales management system developed in C# as part of an Object-Oriented Programming practice project.

The goal of the project is to simulate a basic sales workflow including users, operators, and sales processing using OOP principles.

---

## ⚙️ Features
- Manage users (Korisnik)
- Manage operators (Operator)
- Handle sales transactions (Prodaja)
- Demonstrates OOP concepts:
  - Encapsulation
  - Classes & Objects
  - Basic system modeling

---

## 🧱 Project Structure
- `Program.cs` – Entry point of the application
- `Korisnik.cs` – User model
- `Operator.cs` – Operator model
- `Prodaja.cs` – Sales logic

---

## ▶️ How to Run
1. Open solution in Visual Studio
2. Build solution
3. Run `Program.cs`

---

## 🎯 Purpose of the Project
This project was created to practice:
- Object-Oriented Programming in C#
- Structuring a console application
- Working with multiple classes
- Basic software design principles
